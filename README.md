# Journal

A minimal offline Android journal app built with Jetpack Compose.

## Overview
This app allows users to:
- Add or edit a daily journal entry for the current date only.
- Automatically save as they type.
- View all days with existing entries, sorted by date (newest first).
- Switch between light and dark themes, with the selected theme persisted.

## Tech Stack
- Kotlin
- Jetpack Compose (Material 3)
- Android SDK 26+
- Simple file-based storage

## Current Status
- Basic Compose setup completed
- Core journal logic in progress
- Theme persistence, auto-save, and file handling planned
